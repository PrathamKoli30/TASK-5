# Weather Application

A simple and responsive Weather Application built using **HTML, CSS, and JavaScript**.  
It fetches real-time weather data using the **OpenWeatherMap API** and displays weather conditions with dynamic icons.

main file : (weather.html)

-  Search weather by city name
-  Displays temperature in Celsius
